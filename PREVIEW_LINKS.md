# Southern Oak Claude Preview Links

This repo is a static HTML prototype. It does not need a build step.

Run a local static server from this folder:

```powershell
.\start-preview.ps1
```

Then open:

| Page | URL |
| --- | --- |
| Landing | http://localhost:3003/index.html |
| Site editor | http://localhost:3003/editor.html |
| AI brief | http://localhost:3003/designer.html |
| Projects | http://localhost:3003/projects.html |

Crypto Trader keeps port `3000`; the main Dashlands web app uses `3002`.
